# Task 5: App Optimization, Testing, and Release

## Objective
To write tests, optimize application performance (memory, profiling, and size), implement native device features (e.g. WorkManager, notifications), and generate a signed release APK.

---

## Planned Activities
- **Testing**: Writing unit tests for business logic and UI tests (Compose Tests).
- **Background Tasks**: Integrating WorkManager for periodic background tasks.
- **Push Notifications**: Setting up local/push notifications.
- **Profiling**: Inspecting memory allocations, CPU usage, and frame rate.
- **Release Build**: Generating a signed Android Package (APK/AAB) for distribution.

---

## Deliverables
- Clean, optimized, and fully tested production build.
- Generated APK/AAB package.
- Full internship portfolio and summary documentation.
