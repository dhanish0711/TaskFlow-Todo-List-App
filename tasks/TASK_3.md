# Task 3: Data Persistence and State Management

## Objective
To implement local data persistence in the Android application using Room Database or Preferences DataStore, and orchestrate robust state management using ViewModels.

---

## Planned Activities
- **Room Database Integration**: Creating Entities, DAOs, and database configurations.
- **DataStore Preferences**: Saving simple key-value pairs (like user settings or theme selections) asynchronously.
- **Architecture Pattern**: Implementing Model-View-ViewModel (MVVM) architecture.
- **Coroutines & Flows**: Fetching local database records reactively using Kotlin Flow.

---

## Deliverables
- Fully functional persistent app data storage.
- ViewModel unit tests.
- Task documentation update.
